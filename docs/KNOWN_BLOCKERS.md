# Known Blockers

- Production payment processing requires Stripe credentials and webhook endpoint configuration.
- Production database deployment requires Supabase or equivalent project credentials.
- Production location maps may require Mapbox/Google Maps credentials.
- Mobile distribution requires Apple Developer and Google Play access.
- Push, SMS, and email delivery require vendor accounts and credentials.
- Real POS and club-management sync require vendor access from target golf courses.

Each blocker must be paired with a tested development adapter in its implementation phase before any related requirement can be internally complete.
