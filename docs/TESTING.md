# Testing

Current commands:

- `npm run format` checks repository formatting with Prettier.
- `npm run typecheck` runs strict TypeScript checks.
- `npm run lint` runs the custom repository lint script.
- `npm run test` builds and executes node:test unit tests.
- `npm run validate` runs the full current validation suite.

Future phases must add integration, e2e, RLS, payment, responsive, accessibility, and production-readiness tests.
